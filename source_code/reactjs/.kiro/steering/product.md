# Product Overview

Myanmar Font Preview is a web application for exploring and testing Myanmar Unicode fonts. The app allows users to preview different Myanmar fonts with customizable text, font size, and line height settings.

## Key Features
- Real-time font preview with custom Myanmar text
- Font search and filtering capabilities
- Adjustable typography controls (size, line height)
- Grid/list view toggle
- CSS generation for font integration
- Local storage for user preferences

## Target Users
- Web developers working with Myanmar text
- Designers selecting Myanmar fonts for projects
- Content creators needing Myanmar typography

## Font Categories
The app organizes fonts into four main categories:
- **KhmerType**: Traditional Myanmar fonts (Angoun, Chatu, Phiksel, etc.)
- **Masterpiece**: Modern Myanmar fonts (UniHand, UniRound, UniSerif, etc.)
- **Other**: Standard Unicode fonts (Noto Sans, Pyidaungsu, etc.)
- **Unknown**: Miscellaneous Myanmar fonts
